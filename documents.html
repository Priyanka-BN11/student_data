Student Management API Documentation

1. Get the list of all students

Endpoint:

GET /students

Description:

Returns the list of all students.

Response:

200 OK: Returns an array of student objects.

Example:

[
    {
        "id": 1,
        "name": "Jessica Drake",
        "classes": {
            "biology": 95,
            "algebra": 92
        }
    },
    {
        "id": 2,
        "name": "Ben Cohen",
        "classes": {
            "biology": 95,
            "algebra": 92
        }
    },
    {
        "id": 3,
        "name": "Lisa Downing",
        "classes": {
            "biology": 95,
            "algebra": 92
        }
    }
]

2. Get data about a single student by name

Endpoint:

GET /students/:name

Description:

Returns data about a single student by name.

Parameters:

name (string): The name of the student.

Response:

200 OK: Returns the student object if found.
404 Not Found: If the student is not found.

Example:

{
    "id": 1,
    "name": "Jessica Drake",
    "classes": {
        "biology": 95,
        "algebra": 92
    }
}

3. Add a new student

Endpoint:

POST /students

Description:

Adds a new student to the list.

Request Body:

{
    "name": "New Student",
    "classes": {
        "biology": 90,
        "algebra": 85
    }
}

Response:

201 Created: Returns the added student object with a generated ID.
400 Bad Request: If the name is missing in the request body.

Example:

{
    "id": "generated-uuid",
    "name": "New Student",
    "classes": {
        "biology": 90,
        "algebra": 85
    }
}

4. Delete a student by ID

Endpoint:

DELETE /students/:id

Description:

Deletes a student from the list by ID.

Parameters:

id (string): The ID of the student.

Response:

201 Created: If the student is deleted.
404 Not Found: If the student with the given ID is not found.

Example:

Student generated-uuid was deleted

5. Update the grade of a student by name and class name

Endpoint:

PUT /students/:name/:class/:grade

Description:

Updates the grade of a student by student name and class name.

Parameters:

name (string): The name of the student.
class (string): The class name.
grade (integer): The new grade.

Response:

201 Created: If the grade is updated.
404 Not Found: If the student is not found.

Example:

Student Jessica Drake was assigned a grade of 98 in biology

6. Get the GPA of a student

Endpoint:

GET /students/:name/gpa

Description:

Calculates and returns the GPA of a student by name.

Parameters:

name (string): The name of the student.

Response:

201 Created: Returns the GPA if the student is found.
404 Not Found: If the student is not found.

Example:

3.85
